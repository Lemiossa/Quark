# Quark

## Sobre
Um simples sistema operacional Brasileiro

## Dependências
- build-essential
- mtools
- cross-compiler(i686)

build-essential e mtools creio que esteja no repositorio da sua distro, porem o cross-compiler pode não estar
siga o passo a passo dessa pagina para instalar o cross compiler i686: https://wiki.osdev.org/GCC_Cross-Compiler
